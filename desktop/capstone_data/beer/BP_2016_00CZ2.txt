CB1600CZ21
ZIP Code Business Statistics: Zip Code Business Patterns by Employment Size Class

Release Date : 05/24/2018

NOTE: Data based on the 2016 Zip Business Patterns. For information on confidentiality protection, nonsampling error, and definitions, see  Survey Methodology.

 
 
  Table Name
  ZIP Code Business Statistics: ZIP Code Business Patterns by Employment Size Class: 2016 
  
 
  Release Schedule
  The data in this file were released on April 19, 2018.
  
 
  Key Table Information
  Beginning with reference year 2007, ZBP data are released using the Noise disclosure methodology to protect confidentiality. See  Survey Methodologyfor complete information on the coverage and methodology of the ZIP Code Business Patterns data series. 
  
 
  Universe
  The universe of this file is all operating establishments with one or more paid employees. This universe includes most establishments classified in the North American Industry Classification System (NAICS) Codes 11 through 813990. For specific exclusions and inclusions, see Industry Classification of Establishments. 
  
 
  Geography Coverage
  The data are shown at the 5-digit ZIP Code level only. 
  
 
  Industry Coverage
  The data are shown at the 2- through 6- digit NAICS code levels for all sectors with published data. 
  
 
  Data Items and Other Identifying Records
  This file contains data on the number of establishments. 
  
 
  Sort Order
  Data are presented in ascending ZIP Code sequence.
